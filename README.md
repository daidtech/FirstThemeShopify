# FirstThemeShopify

FirstThemeShopify is a personal practice Shopify theme built to learn **Liquid**, **Dawn architecture**, and **Online Store 2.0**.

## Purpose
This project is for learning and experimentation only. It helps me understand how Shopify themes work internally and how to build custom UI using Shopify best practices.

## What I Practice
- Shopify Liquid basics (`{{ }}` and `{% %}`)
- Dawn theme structure (layout, templates, sections, blocks)
- Creating custom pages and sections
- Online Store 2.0 concepts
- Responsive storefront UI

## Tech Stack
- Shopify
- Liquid
- HTML / CSS
- JavaScript (basic)
- Dawn theme
